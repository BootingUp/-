## 三焦

`三焦，为六腑之一，是上、中、下三焦的合称。将人分割三部分，横膈以上内脏器官为上焦，包括心、肺；横膈以下至脐内脏器官为中焦，包括脾、胃、肝、胆等内脏；脐以下内脏器官为下焦，包括肾、大肠、小肠、膀胱。`

## 五邪

`风、寒、湿、燥、热，分外生和内生`

### 风
`风为阳邪，其性轻扬升散，具有升发、向上、向外的特性`  
`风与肝相应。风为木气，通于肝。 外感风邪可导致胃脘痛、腹胀、肠鸣、呕吐、泄泻等。这是风邪伤肝，木盛克土所致。`
